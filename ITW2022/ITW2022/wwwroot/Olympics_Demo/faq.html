Olimpics API is a website that wants to help you better know the Olimpic Games, by organizing all editions, athletes and competitions in a manageable and meaningful way. We do this by the way of tables, various search engines and charts. Using this tool will allow you to pinpoint that specific athlete, edition or competition you've been looking for.
